# TradeGuard Mobile

This folder contains a Flutter skeleton for the TradeGuard MVP.

Current screens:

- Home
- Credit result
- Evidence upload
- Certificate result

Flutter CLI is not installed on this machine, so the project was scaffolded manually and not compiled yet.

Recommended next steps on a Flutter-enabled machine:

1. `flutter create .`
2. merge or preserve the existing `lib/` files
3. run `flutter pub get`
4. connect the app to the backend API
